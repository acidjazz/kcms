
CMS JSON Project
===
Dynamically generated form based on associative multi-dimensional JSON data


### Requirements (advised to install in order)

* Mongo DB 2.4.x + 
* PHP 5.4+
    * Mongo Driver
    * OpenSSL 
    * PHP-FPM (for nginx communication)
* Nginx 1.4.x 
* Node.js v0.10.x +
    * npm 1.4.x +
    * jade 1.2.x +
    * stylus 0.4x +
